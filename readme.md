# 🃏 Card Game

A simple and interactive card game built with **Next.js** and **Firebase Hosting**.  
This project demonstrates frontend game logic, component-based UI, and automated deployment with GitHub Actions and Firebase.

---

## 🌐 Live Demo

👉 **[Play the game here!](https://card-game-8e7cb.web.app/)**  
---

## 🚀 Features

- 🎮 Interactive card gameplay  
- 💥 Real-time score updates  
- 🎨 Clean and responsive UI  
- 🔥 Deployed automatically via **Firebase Hosting + GitHub Actions**  
- 🧠 Built with **Next.js 14** and **TypeScript**

---

## 🛠️ Tech Stack

| Tool / Library | Purpose |
|-----------------|----------|
| **Next.js** | Frontend framework |
| **React** | UI components |
| **TypeScript** | Type safety |
| **CSS / Tailwind / Custom CSS** | Styling |
| **Firebase Hosting** | Deployment platform |
| **GitHub Actions** | CI/CD for auto deploys |
